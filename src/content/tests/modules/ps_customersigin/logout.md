# Logout

## Preconditions

Go to back office -> Modules -> Module manager<br />
Search ps_customersignin<br />
The module should be installed and upgraded to the last version<br />
Go to FO
## Steps
### Step #0
#### Description
In front office go to login
#### Expected result
Login page should be displayed
### Step #1
#### Description
Click on logout
#### Expected result
FO should be logout
