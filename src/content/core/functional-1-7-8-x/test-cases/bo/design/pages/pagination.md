## Steps

Catégories Pages\

-> Aller dans pages 
-> Cliquer sur ajouter une nouvelle catégorie de page
-> Répéter l'action 11 fois
-> Changer le nombre d'item de 50 à 10 par page
-> Cliquer sur suivant
-> Cliquer sur précédent\
\
Pages\

-> Aller dans pages 
-> Cliquer sur ajouter une nouvelle page
-> Répéter l'action 11 fois
-> Changer le nombre d'item de 50 à 10 par page
-> Cliquer sur suivant\
